
**- About:-**
Job Tracker is a React-based application designed to help users manage and keep track of recent job openings. The app allows users to view detailed information about each job, including the job title, company, location, salary, and job description. Users can easily add new job listings, edit existing ones, or delete jobs that are no longer relevant.
